Pseudoclases

Estilos especiales dependiendo del contexto, posición o estado.

:root

Pseudoclase que selecciona el elemento raíz de un documento. Identico al selector "html" excepto que su nivel de especificidad es mayor. También usado para crear variables.

:hover

Pseudoclase que se presenta cuando el usuario coloca el puntero sobre algún elemento, pero no necesariamente está activo.

:active

Pseudoclase que cubre el caso en el que un elemento está siendo activado por el usuario. Cuando se interactua con el ratón ... el momento en el que el usuario presiona el botón del ratoón hasta el momento en el que lo suelta. A menudo usado en elementos.

    El orden SI importa:
  1. :link { ... }
  2. :visited { ... }
  3. :hover { ... }
  4. :active { ... }

:target

Pseudoclase representa el elemento único, si existe alguno, con un id coincidente con el identificador de fragmentos de la URL del documento.

:nth-child :first-child :last-child

  • ntr-child: seleciona el elemento indicado que sea el enésimo hijo de su elemento padre.
  • first-child: seleciona el elemento indicado que sea el primer hijo de su elemento padre.
  • last-child: seleciona el elemento indicado que sea el último hijo de su elemento padre.

:nth-of-type :first-of-type :last-of-type

  • nth-of-type: seleciona el elemento indicado que sea el enésimo hijo en su tipo sin importar la jerarquia dentro del elemento padre.
  • first-of-type: seleciona el elemento indicado que sea el primer hijo en su tipo sin importar la jerarquia dentro del elemento padre.
  • last-of-type: seleciona el elemento indicado que sea el último hijo en su tipo sin importar la jerarquia dentro del elemento padre.

:not

Pseudoclase de negación, :not(X) es una notación funcional que toma un selector simple X como argumento. Coincide con un elemento que no está representado por el argumento. X no debe contener otro selector de negación

:empty

Pseudoclase corresponde a un elemento sin ningún nodeo hijo. Sólo cuando hay nodos de elemento o texto con uno o más caracteres dentro del elemento ése de convierte en no vacío.
